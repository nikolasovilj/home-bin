# home-bin
## set of utilites I wrote

Add home-bin to your PATH variable so that utils are accessible 




```sh
echo export PATH=\$PATH:/home/{username}/home-bin >> .bashrc

cd ~ && git clone https://github.com/nikolasovilj/home-bin.git && user=`whoami` && echo export PATH=\$PATH:/$user/home-bin >> .bashrc
```
